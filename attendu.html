ATTENDU

Revolutionary way organization of events

Attendu is a revolutionary platform that combines cutting-edge technology with a human approach to transform event organization. As a solo UX/UI designer, I was responsible for designing intuitive interfaces and developing a strong brand identity for Attendu's web and mobile applications.

Challenges

  • User-Centered Design: Designing a platform that caters to the diverse needs of event organizers and attendees while providing a seamless experience for all users.
  • Complex Information Architecture: Organizing a vast amount of event data in a way that is intuitive, accessible, and visually appealing.
  • Seamless Cross-Platform Experience: Creating a consistent and user-friendly experience across web and mobile platforms.

Solutions

  • Adaptive Design System: I created an adaptive design system that ensures a seamless cross-platform experience. This involved developing reusable UI components and patterns that easily adapt to different screens, ensuring consistency and a cohesive user experience across web and mobile applications.
  • Information Hierarchy and Navigation: I established a clear information hierarchy, intuitive navigation, and search functionalities to help users find relevant information easily.
  • Seamless Cross-Platform Experience: Creating a consistent and user-friendly experience across web and mobile platforms.

Overview

Branding

I developed a strong brand identity for Attendu that includes a distinctive logo, a color palette, and a set of typography that are all consistent with the platform's user interface. The brand identity also includes a set of messaging that conveys the platform's value proposition to event organizers and attendees.

web app

Page & Email Builder

I designed a page and email builder for Attendu that allows event organizers to create and customize pages and emails for their events. The page and email builder includes a variety of features, such as drag-and-drop functionality, a library of pre-made templates, and the ability to preview changes before publishing. The page and email builder has helped to simplify the process of creating and managing event pages and emails for event organizers.

Guest App

I designed a guest mobile app for Attendu that allows attendees to easily access information about their events, such as the event schedule, speaker bios, and maps. The guest mobile app also includes features such as check-in, messaging, and event feedback. The guest mobile app has helped to improve the experience of attending events for attendees.
